# php-code-review-sniff

A code review sniff built for PHP Code Sniffer.

This tool will cover:

- PHP Patterns
    - PSR-1
    - PSR-2
- Docblocks
    - If they are included for:
        - Classes
        - Methods
        - Attributes
        - If they apply the right tags for each content
- Tests
    - If they exist

This tool will not cover:

- Views
    - twig, phtml
